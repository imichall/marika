## Jak přispívat do changelogu

1. Pro každou významnou změnu vytvořte nový záznam v příslušné sekci (main nebo dev)
2. Uveďte:
   - Verzi (commit SHA)
   - Popis změny
   - Autora
   - Datum
3. Dodržujte formát Markdown pro lepší čitelnost
